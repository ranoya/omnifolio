Ponto de quebra

Os media queries (@media no CSS) são formas de definir pontos no dimensionamento onde todo o layout precisa ser alterado. Neste exemplo, o ponto de quebra é de 800 pixels na horizontal. Quando o navegador atinge uma dimensão menor do que esta, o layout se altera para que os quadros assumam a totalidade do espaço disponível. Os pontos de quebra são necessários para que o conteúdo permaneça legível em qualquer condição de uso, impedindo que um quadro fique muito pequeno, ou até grande demais.

A composição, logicamente, precisa ser pensada para lidar com este tipo de situação (e/ou tirar partido dela).
