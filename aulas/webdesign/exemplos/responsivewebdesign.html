Este é um texto completo, que contém toda a informação que deveria constar na página, explicando detalhadamente o que é a informação sendo exibida. Quando o contexto é mais amplo, este texto aparece e ocupa o espaço destinado para o conteúdo; quanto o contexto é reduzido, um outro conteúdo aparece em seu lugar, mais enxuto e sintético, desenvolvido para quem está usando este documento em uma plataforma mobile. Não faz sentido apresentar laudas de texto em um dispositivo que não foi criado para a leitura.
